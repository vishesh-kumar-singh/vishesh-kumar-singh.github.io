Gender Detection Model using CNN↗

December 2024

I developed a Gender Classification Model using Convolutional Neural Networks (CNNs) as part of the winter project offered by the IIT Kanpur Consulting Group. The model achieved an impressive 95.499% accuracy on a test dataset of 1000 images.


Key Features:

  • Almost Accurate Gender Prediction: Classifies the gender of all faces detected in an image.
  • Real-Time Predictions: Performs gender classification via live video capture using a webcam.


Methodology:

  • Designed and trained a CNN on a balanced dataset of 8000 images.
  • Integrated OpenCV Haar-Cascade for detecting faces in static images from the internet, enabling gender predictions.
  • Used cvlib for real-time face detection and live gender classification via webcam.


This project was a rewarding experience that enhanced my skills in deep learning, computer vision, and real-time application development.

J.A.R.V.I.S.↗

November 2024

A voice-enabled personal assistant with functionalities like text-to-speech, note-taking, file handling, and interactive games. This project was inspired by the J.A.R.V.I.S. AI from the Iron Man movies and aimed to create a versatile and interactive assistant.


Key Features:

  • Text-to-Speech: Converts text input into spoken words, providing an auditory output for various tasks.
  • Note-Taking: Allows users to create and save notes just by voice command.
  • Interactive Games: Includes simple voice-controlled games like Rock-Paper-Scissors for entertainment and engagement. The high score is also saved to increase interactivity.


Methodology:

  • Utilized Python libraries such as SpeechRecognition for voice input and pyttsx3 for text-to-speech conversion.
  • Used loops and conditional statements for executing commands by detecting particular keywords in commands.


This project was a comprehensive exercise in integrating various technologies to create a cohesive and functional personal assistant. It enhanced my skills in Python software development.
