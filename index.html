Publications

Biological Memory Consolidation in SNNs Under Revision for Elsevier Neurocomputing
Code DOI

Spiking Neural Networks (SNNs) offer a promising paradigm for energy-efficient neuromorphic computing, yet maintaining knowledge across sequential tasks remains a significant challenge. This work explores a biologically grounded method for long-term memory preservation through synaptic consolidation. We introduce a Potentiation-Factor (P-Factor) to identify neural patterns responsible for predictions, enabling selective memory preservation during new task learning.

Dual-Phase Learning:

  • Synaptic Locking: Freezing of identified expert neurons to preserve learned representations
  • Homeostatic Reset: Recycling of redundant novice neurons to maintain plasticity for new tasks

Results: Benchmarked on Split-MNIST, Split-N-MNIST & 5-Split-Fashion-MNIST. Superior performance over all traditional methods (EWC, SI, PackNet) adapted for SNNs, surpassing by more than 30%.

Dynamic Distractor Pruning in Deep RL (EMA-Tree SAC) Under Review for TMLR
Code

Stabilize continuous control RL agents by dynamically filtering high-variance noise and spurious distractors.

  • Developed EMA-Tree SAC, leveraging Random Forests to extract predictive importance from Q-values.
  • Stabilized selection via Exponential Moving Average (EMA) to create a dynamic binary hard-mask.
  • Engineered an adversarial Distracting Control Suite injecting Gaussian noise and coupled spurious states.
  • Outperformed L1 and Group-Lasso regularised SAC, proving hard-masking beats standard weight-decay.

Impact: Eliminated destructive noise on-the-fly, yielding up to a 51% return improvement over standard SAC.
